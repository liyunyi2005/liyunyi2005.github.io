# 我的第一篇博客文章

欢迎来到我的博客！这是使用 Markdown 编写的示例文章。

## 文章内容

你可以在这里编写你的博客内容，支持：

- **粗体文本**
- *斜体文本*
- `代码块`
- [链接](https://example.com)
- 列表项

### 代码示例

```javascript
console.log('Hello, World!');
```

### 图片

如果需要添加图片，可以将图片放在 `src/assets/images/` 目录下，然后引用：

```markdown
![图片描述](./image.png)
```

祝你写作愉快！
